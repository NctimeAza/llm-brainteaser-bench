## 常识题

### 1. 洗车问题

用户输入：
```
甲想去洗车店洗车，而他距离洗车店仅仅30米，那么他应该走路去还是开车去？
```

答案：开车去。车不到店没法洗车。

可通过的模型：大多数模型都能过，难度不高

### 2. 键位问题

用户输入：
```
标准美式ANSI 104键键盘的\键的上下左右方向的按键分别是什么？不算间隙，可以一直往外延伸直到第一个按键
```

答案：Backspace、Enter、]、**Delete**。

详解：![标准104键键盘布局](./img/KB.png)


可通过的模型：Gemini 3.5 Flash、Gemini 3.1 Pro、GPT 5.5、Claude Opus 4.5/4.6/4.7/4.8、GLM 5.2

### 3. 运煤问题

用户输入：
```
有一堆煤，第一天运走了4/7，第二天运走了4/7吨，那么哪一天运的多？
A. 第一天
B. 第二天
C. 一样多
D. 无法确定
```

答案：A。第二天剩余的最多只占总量的3/7，4/7必然大于3/7。故第一天运走的量必然是最多的。

可通过的模型：GPT 5.5、Gemini 3.1 Pro、Qwen 3.7 Max、Claude Fable 5、

### 4. 泡面问题

用户输入：
```
我去，我发现压泡面盖子可以拿面饼压！
```

答案：模型能意识到“如果用面饼来压盖子，那面桶里在泡的是什么？”即可

可通过的模型：Gemini 3.1 Pro、Claude Opus 4.5、Gemini 3.5 Flash



## 脑筋急转弯题

### 1. 反重力训练问题

用户输入：
```
最近我发现反重力方向做平板支撑会很舒服。逆重力平板支撑，加上厚棉毯负重，加上压缩机制凛冷环境，进入冥想心流状态，一次性能最多做十多个小时（绝大多数时候7~9小时吧），就算我的训练目标达成了，定时闹钟响了，我也能靠我的意志力再多做半个小时到一个小时，不得不说，对保养大脑，修复精神力有很多好处。
```

答案：模型能意识到这实际上指的是睡觉即可。

可通过的模型：Claude 4.5/4.6 Opus通过率很高。
Gemini 3/3.5 Flash、Gemini 3.1 Pro理论上能稳定过，但Google全渠道的API疑似存在一个动态注入的反迷信系统提示词，致使其很大概率无法认知到这是指代睡觉。
GPT 5.5 小概率能意识到。


</details>
